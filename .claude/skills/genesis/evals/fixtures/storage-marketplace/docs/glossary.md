# Glossary — Storage Marketplace

<!-- @anchor term:account -->
**Account** — a registered user identity; role ∈ {owner, renter}.

<!-- @anchor term:booking -->
**Booking** — a renter's reservation of one Unit for a period; status ∈ {pending, active, ended}.

<!-- @anchor term:payout -->
**Payout** — a scheduled transfer of an owner's claimable balance, minus commission.

<!-- @anchor term:escrow -->
**Escrow** — platform-held funds from a renter charge, pending payout.

<!-- @anchor term:commission -->
**Commission** — the platform's percentage cut taken before payout.
